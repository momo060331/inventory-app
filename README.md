# 家庭在庫管理アプリ

Firebase連携による家族共有型の在庫管理ウェブアプリです。

## 特徴

- 📱 PWA対応 - スマホ/PCにアイコン付きでインストール可能
- 🔄 リアルタイム同期 - 複数デバイス間でデータを自動同期
- 📷 バーコードスキャン対応
- 👥 プロファイル機能 - 家族それぞれの表示カスタマイズ
- 📊 在庫アラート機能
- 💾 オフライン対応

## セットアップ手順

### 1. GitHubでの公開

1. このリポジトリをフォークまたはクローン
2. リポジトリの Settings > Pages を開く
3. Source を "main" ブランチに設定
4. 数分待つとURLが表示されます（例: `https://yourusername.github.io/inventory-app/`）

### 2. アプリのインストール

#### スマートフォン（iOS/Android）

1. 公開されたURLをブラウザで開く
2. **iOS**: Safariの共有ボタン → 「ホーム画面に追加」
3. **Android**: Chromeのメニュー → 「ホーム画面に追加」

#### PC（Chrome/Edge）

1. 公開されたURLをブラウザで開く
2. アドレスバー右端のインストールアイコンをクリック
3. 「インストール」をクリック

### 3. アイコン画像の設置

`icon-192.png` と `icon-512.png` を用意してリポジトリのルートに配置してください。

簡易的なアイコン生成方法：
- オンラインツール（例: https://www.favicon-generator.org/）を使用
- 📦 絵文字をスクリーンショットして使用
- 画像編集ソフトで作成

```
inventory-app/

├── service-worker.js    # オフライン対応
├── icon-192.png         # アプリアイコン（小）
├── icon-512.png         # アプリアイコン（大）
└── README.md           # このファイル
```

## Firebaseの設定

現在のコードには既にFirebase設定が含まれていますが、本番利用の場合は独自のFirebaseプロジェクトを作成することを推奨します：

1. [Firebase Console](https://console.firebase.google.com/) でプロジェクトを作成
2. Firestore Databaseを有効化
3. ウェブアプリを追加して設定情報を取得
4. `index.html` の `firebaseConfig` 部分を書き換え

## 使い方

1. 初回起動時にプロファイルを作成
2. カテゴリーを設定（または既存のものを使用）
3. 商品を登録（バーコードスキャンまたは手動入力）
4. 在庫を管理・確認

詳しい使い方はアプリ内の各機能をご確認ください。
